# MathP

Egy projekt a Pannonos tanulmányaimhoz, hogy szerkesztőn belül kóddal tudjak számolni.

## Funkciók

- Vektorok
  - műveletek (+ - * /)
- Mátrixok
  - műveletek (+ - * /)
  - mátrix szorzás
  - transzponálás
  - determináns számítás 
- Gráfok
  - topológiai rendezés
- Egyéb eszközök
  - futási időt figyelő óra
